요청 위임된 request에 존재하는 파라미터 출력하기

        ${param.key}
            - request에 존재하는 parameter 값 얻어와 출력함
  • 제출된 이름
  • 제출된 나이
  • 체크된 색상

th:text / th:utext

        th:text = "속성값"
            해당 태그에 "속성값"을 내용으로 출력하지만 html태그, 특수문자 해석 X (innerText)

        th:utext= "속성값"
            해당 태그에 "속성값"을 내용으로 출력
            html태그와 특수문자 해석 O (innerHTML)

th:text (HTML 태그 해석 불가능)
